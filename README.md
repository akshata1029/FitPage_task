Test Automation Setup & Execution
Setup
1.Install Python.
Install required packages using:
pip install -r requirements.txt
Ensure WebDriver (e.g., ChromeDriver) is available.
Running Tests
To run all tests:
pytest
To run a specific test file:
pytest Tests/test_event.py
